# Photo of the Day

* Downloads Bing Photo of the Day and makes it desktop background
* Saves all photos to ~/Pictures/Bing
* Supports different localized photos with -l or --locale arguments

## Supported Desktops

* Mac OS X
* Linux Unity
* Linux GNOME
